# Restaurant Tip Analysis

## Project Overview
This project explores tipping behavior in restaurants using the Seaborn "tips" dataset.

## Dataset
The dataset contains information about restaurant bills and tips including:
- total bill
- tip
- gender
- smoker
- day
- time
- party size

## Question
Is there a relationship between the total bill and the tip amount?

## Analysis
A scatter plot was used to visualize the relationship between the total bill and tip.

## Insight
The analysis shows a positive correlation between bill size and tip amount.
Customers generally tip more when the total bill is higher.

## Tools
- Python
- Pandas
- Seaborn
- Matplotlib
